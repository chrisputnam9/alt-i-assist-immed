# Overview

## Hackathon:
https://geminiliveagentchallenge.devpost.com/

## Idea: Alt-I - AI Everywhere

### Chrome Extension with:
 - All permissions needed for the full plan
 - Iconography based on what is in assets folder

### Tracer Bullet Features:
 - Configuration page and UX help for setting up Gemini Live API key
 - Hit Alt-I anywhere (globally, not just in Chrome)
 - Using Gemini Live API, you can talk and get answers
 - Long-term, we will give the AI more and more context and capabilities

## Development Tools:
 - Auto-reload for local development
 - Prettier
 - Linting with well-regarded standard rules
 - Use Typescript
 - Github action to package and publish to Chrome Web Store on push to master
 - Github action to check linting
