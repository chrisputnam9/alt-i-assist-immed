Alt-I Assist Settings

Quick Start Guide:
  • Press Alt + I from anywhere to toggle the Voice Assistant.
  • Wait for the Start Tone and the extension badge to show ON.
  • Speak naturally. The assistant is listening and will respond in real-time.
  • Press Alt + I again to deactivate.

Gemini API Key

To use this extension, you need a Gemini API Key from Google AI Studio.

  1. Visit Google AI Studio API Keys.
  2. Generate a new API key.
  3. Paste the key in the field below and click Save.
