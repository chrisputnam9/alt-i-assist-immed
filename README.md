# Alt-I Assist. Immed.

**AI Everywhere.** Push-to-Talk with your Live Assistant.

Hit a global keyboard shortcut (`Alt+I`), start talking, and get immediate multimodal assistance. **Alt-I** leverages the Gemini Multimodal Live API to provide low-latency, voice-to-voice interaction from anywhere on your system.

## Project Overview

This project is an entry for the [Gemini Live Agent Challenge](https://geminiliveagentchallenge.devpost.com/). It aims to demonstrate the power of real-time AI agents integrated directly into the user's workflow via a Chrome Extension.

- **Category:** Live Agent (Real-time Audio/Vision interaction)
- **Tech Stack:** TypeScript, Vite, Chrome Extension (Manifest V3), Gemini Multimodal Live API.
